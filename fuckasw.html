How to use

  1. Download BSManager and install your favorite version of Beat Saber

  2. Download FuckASW.bat

  3. Create a shortcut using the gear icon on the top right
    Make sure to tick "Create a Steam shortcut"


  4. Open Steam and copy the launch options from the shortcut you just made

    5. If you don't see the new shortcut, try restarting Steam

  5. Right click "FuckASW.bat" and hit Edit

  6. Paste the launch options in place of "PASTE HERE" (without removing the quotation marks)

  7. Save the file and close it

  8. Open "FuckASW.bat" and there you go :3
